# EasyBluetooth-community

Community hub for EasyBluetooth tutorials, unified data interface documentation, integration examples, device adaptation tasks, and selected open-source helper tools.

EasyBluetooth application and official providers are NOT open source.
This repository contains public documentation, specifications, community resources, and companion helper tooling only.

Commercial integration with EasyBluetooth requires authorization.

- AIDA64 Helper source: `tools/Aida64Helper/`
